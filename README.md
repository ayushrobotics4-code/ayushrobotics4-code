🚀 About Me



​I am a Robotics Enthusiast & Developer dedicated to building functional, interactive hardware. My primary focus is on Embedded Systems and Autonomous Robotics, specifically using Arduino Nano and ESP32 microcontrollers. I specialize in integrating hardware components like Bluetooth modules, servo drivers, and sensors (IR, Ultrasonic) to create responsive, real-world machines.
​I leverage AI-assisted development via Gemini and Claude to streamline my workflow, while doing the heavy lifting in VS Code and Antigravity. My work bridges the gap between hardware and software, from circuit design to custom web dashboards.🤖🤖
